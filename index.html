How to Use

  1. Install the extension from Firefox Add-ons Store
  2. Open any YouTube video
  3. Click "Off" to switch to "On"
  4. Set the audio delay using the buttons +/-500 +/-100
  5. Press Reset to set to 0
  6. Click the add-on icon. In the add-on settings you can change the panel's appearance and set the default delay
